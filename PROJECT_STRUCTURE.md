# Структура проекта

```text
src/
├── app/              # маршруты App Router
├── components/       # layout, sections, forms, services, fleet, documents, ui
├── content/          # RU/EN словари
├── data/             # редактируемые данные сайта
├── hooks/            # localStorage state
├── lib/              # storage, requests, analytics, paths
└── types/            # типы сайта
```

Публичные ассеты лежат в `public/`: изображения, PDF, favicon, manifest, sitemap, robots, CNAME.example.
