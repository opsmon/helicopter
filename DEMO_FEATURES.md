# Функции демо

## Работает полностью в демо

- Статические страницы и адаптивная навигация.
- Мобильное меню с Escape и блокировкой фоновой прокрутки.
- Быстрая и универсальная формы заявок.
- Zod-валидация.
- Черновик заявки в localStorage.
- История заявок и страница статуса.
- Избранное, сравнение и история просмотров услуг.
- Поиск, фильтр, сортировка, открытие и скачивание документов.
- Web Share API с fallback на копирование ссылки.
- Печать заявок, документов, контактов и пассажирских правил.

## Имитируется

- Отправка заявки менеджеру.
- Номер заявки.
- Статусы заявки.
- Характеристики флота.
- Кейсы и маршруты.

## Требует backend

- Реальная передача заявки в CRM.
- Загрузка файлов.
- Авторизация.
- Личный кабинет.
- Онлайн-оплата.
- Уведомления SMS/Telegram/email.
